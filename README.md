# يُسر — الموقع التعريفي

صفحة هبوط (Landing Page) احترافية لتطبيق **يُسر** المالي:
دفترك الرقمي لإدارة الالتزامات والديون، بطاقة رقمية بتقنية NFC، مدفوعات وتحويلات فورية، وتقسيط يناسبك.

## التقنيات
- HTML5 + CSS3 (تصميم متجاوب بالكامل، RTL، Flat / Modern)
- JavaScript خالص (بدون مكتبات): قائمة جوال، Scrollspy، عدّادات متحركة، أكورديون للأسئلة، حركات الظهور عند التمرير، نموذج تواصل.
- خط Cairo و Font Awesome.

## البنية
```
index.html        الصفحة الرئيسية
css/styles.css     التصميم وهوية يُسر (الأخضر الزيتوني)
js/script.js       التفاعلات
assets/img/        الشعار ولقطات التطبيق
```

## التشغيل
افتح `index.html` مباشرة في المتصفح، أو شغّل خادماً محلياً:
```bash
python3 -m http.server 8000
```
ثم زر <http://localhost:8000>.
