# يُسر — الموقع التعريفي

صفحة هبوط (Landing Page) احترافية لتطبيق **يُسر**:
وسيط رقمي آمن لتوثيق الديون والعربون عبر تقنية NFC — يحوّل الدفاتر الورقية إلى سجلات
موثّقة غير قابلة للتلاعب تحفظ الحقوق وتمنع النزاعات — وسيط توثيق يوثّق الحق الآن، والسداد يكون لاحقاً بمرونة.

## ماذا يقدّم يُسر
- **الحدّ الائتماني الموحّد** — قاعدة بيانات موحّدة لكل المتاجر تحميها من الاحتيال المالي: لا أحد يتديّن بأسماء غيره أو يتنقّل بين المتاجر متراكماً عليه الديون.
- **توثيق فوري عبر NFC** (تطبيق-إلى-تطبيق) للديون والعربون.
- **الدفتر الرقمي** وتوثيق العربون والعقود بختم زمني.
- **المساعد الذكي (AI)** و**المسح الذكي (OCR)** لاستخراج وتوثيق التفاصيل تلقائياً.
- **أرشفة وتصدير PDF موثّق** وتذكيرات سداد.
- **خيارات سداد مرنة** — وثّق الآن وادفع لاحقاً مباشرةً، أو قسّط عبر شركاء (التاجر يستلم مستحقاته دفعة واحدة شهرياً)، أو الجمع بينهما.

## الباقات
- **الأفراد:** مجاني بالكامل.
- **التجار:** الأساسية 19 ر.س/شهر · المتقدمة 39 ر.س/شهر · الاحترافية 79 ر.س/شهر (NFC في جميعها).
- **المؤسسات والبنوك (B2B):** ترخيص API (White-label) ومصرفية مفتوحة (Open Banking) وفق إطار ساما.

## التقنيات
- HTML5 + CSS3 (متجاوب بالكامل، RTL، تصميم عصري).
- JavaScript خالص: قائمة جوال، Scrollspy، عدّادات، أكورديون، حركات الظهور، نموذج تواصل.
- خط Cairo و Font Awesome.

## البنية
```
index.html        الصفحة الرئيسية
css/styles.css     التصميم وهوية يُسر (الأخضر الزيتوني)
js/script.js       التفاعلات
assets/img/        الشعار ولقطات التطبيق
```

